Ferramentas utilizadas

  • length: retorna a quantidade de elementos em uma lista
  • push: adiciona um valor ao final de uma lista
  • sort: organiza uma lista em uma determinada ordem
  • document.querySelector: busca um elemento HTML na árvore DOM pelo seu seletor
  • document.querySelectorAll: busca todos os elementos HTML na árvore DOM pelo seu seletor
  • document.createElement: cria um elemento DOM HTML
  • elemento.setAttribute: adiciona um atributo HTML a um elemento
  • elemento.getAttribute: recupera o valor de um atributo HTML do elemento
  • elemento.appendChild: adiciona um elemento DOM como descendente de outro elemento DOM
  • elemento.addEventListener: "escuta" por um evento e dispara uma função
  • elemento.removeEventListener: remove a função relacionada a um evento de um elemento
  • alert: exibe uma janela de alerta do navegador
  • setTimeout: agenda uma execução para um tempo posterior
